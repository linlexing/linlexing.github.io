国庆大门岛
========================

[上篇](/2016/10/01/国庆乐清.html)

2016-10-02
------------------------
今年大门大桥正式通车了，大门岛隶属洞头县，我们都称为"下山"，以前都要从黄华码头乘渡轮，现在通车了，我还没有去过，于是决定去玩一趟。

临出门前，拿了一顶爸爸戴的竹笠，戴上试试，我怎么想起了港片中的东厂太监，好像也是这个造型。

![]( {{ site.url }}/assets/blog-images/20161002/a-21.jpg)

换个表情，这下顺眼多了，像不像初出茅庐的侠客

![]( {{ site.url }}/assets/blog-images/20161002/a-22.jpg)

大门大桥从翁垟上去，一路上风景很不错，远远就望见一座云雾缭绕的山头，那就是大门岛

![]( {{ site.url }}/assets/blog-images/20161002/a-26.jpg)

刚下桥，就迫不及待地靠边停下，那边山头云雾在日光的照耀下，有着梦幻般的效果，可惜镜头还是太短，够不着

![]( {{ site.url }}/assets/blog-images/20161002/a-23.jpg)

停车的位置好像是个码头，一艘渡轮正在起航

![]( {{ site.url }}/assets/blog-images/20161002/a-24.jpg)

这梦幻般的，怎么拍都不够

![]( {{ site.url }}/assets/blog-images/20161002/a-25.jpg)

继续向前，路过一个观景台，路边有买沙滩玩具的，后头的山上雾气腾腾，山不在高，有气则灵

![]( {{ site.url }}/assets/blog-images/20161002/a-26.jpg)

从观景台往下眺望，一副标准的温州农村景象，能造房子的地方都造上了

![]( {{ site.url }}/assets/blog-images/20161002/a-29.jpg)

远处就是一望无际的大海了，今天的天气很不错，晴空万里无台风，但这个大太阳，让我暗暗担心，这可不是出照片的天气

![]( {{ site.url }}/assets/blog-images/20161002/a-30.jpg)

这里风光无限，是个合影的好地方

![]( {{ site.url }}/assets/blog-images/20161002/a-31.jpg)
![]( {{ site.url }}/assets/blog-images/20161002/a-33.jpg)
![]( {{ site.url }}/assets/blog-images/20161002/a-34.jpg)

临走前，下意识地拍了一张，后期整理照片时，效果让我非常惊艳

![]( {{ site.url }}/assets/blog-images/20161002/a-32.jpg)

终于到了目的地了，马岙潭沙滩，大家都很兴奋，虽然太阳很大

![]( {{ site.url }}/assets/blog-images/20161002/a-35.jpg)

然并卵，沙滩呢，只有海水，不见一粒沙子

![]( {{ site.url }}/assets/blog-images/20161002/a-36.jpg)
![]( {{ site.url }}/assets/blog-images/20161002/a-40.jpg)

好嘛，原来现在在涨潮，沙滩在水下面，这玩沙滩也得像看钱江潮一样，得掐着点去。

大老远的都到了，能怎么办，只能等着呗。边上有租帐篷的，100元一顶

![]( {{ site.url }}/assets/blog-images/20161002/a-38.jpg)

还有卖小吃灯盏糕的

![]( {{ site.url }}/assets/blog-images/20161002/a-39.jpg)

等着无聊就边上逛逛，大门岛就是一座山峰，而且是石头山峰，没有多少平地，房子大都是照在山坡上

![]( {{ site.url }}/assets/blog-images/20161002/a-41.jpg)

这里的海水倒是挺清的，比舟山朱家尖好

![]( {{ site.url }}/assets/blog-images/20161002/a-42.jpg)

被海水淹没的礁石旁，看到了马岙潭的题字

![]( {{ site.url }}/assets/blog-images/20161002/a-43.jpg)

在山脚下海边有着一片乱石滩，看到了很多漂浮过来的垃圾，其中就有很多台风打过来的木头，听本地人讲，上个台风打走了两米的沙滩，所以今天一点也看不到了。

![]( {{ site.url }}/assets/blog-images/20161002/a-44.jpg)

无聊地只能看看海水

![]( {{ site.url }}/assets/blog-images/20161002/a-45.jpg)

这时，二哥拔了一株乌饭子给我们吃，路过的一个本地人看了说，这东西好吃的很，小时候他们就是专门吃这个当零食。但是我吃着怎么就像吃棉花球，一点点酸

![]( {{ site.url }}/assets/blog-images/20161002/a-46.jpg)

临近中午，人越来越多，我们决定离开先去吃饭，在来的路上，看到有几家饭店，于是步行走回去。从高处眺望，风景好多了，那边好像是玉环

![]( {{ site.url }}/assets/blog-images/20161002/a-47.jpg)

好多渔船停泊在这里，听二哥讲，我们的老太公就是在这里打渔发家的

![]( {{ site.url }}/assets/blog-images/20161002/a-48.jpg)

路边有晒鳗干的

![]( {{ site.url }}/assets/blog-images/20161002/a-49.jpg)

下面海面上很多不明物体，好像是养紫菜的

![]( {{ site.url }}/assets/blog-images/20161002/a-50.jpg)

我们好不容易走到了饭店，却发现被包场了，办婚宴，然后问问人，说是边上没有吃饭的地，这真是坑死人。我们一路走上来，发现路边停了几公里的车，想想下面的沙滩才这么点大，人太多了，想不到这么个不知名的小岛也会闹十一，于是决定提早离开，去大门镇吃饭去。

这里的山路只能两车通过，停了一溜的车，只剩下单车道，然后就开始拥堵了，我们把车开出来，花了足足二十分钟。

出来后，顺着弯弯的山路，一路到了大门镇，已经是下午一点半了，然后路边看到一家小饭店，赶紧停下吃饭，从十一点多开始找吃饭的地方，到现在，两个小时了才吃上饭。

![]( {{ site.url }}/assets/blog-images/20161002/a-53.jpg)

这里是一座石山岛，最不缺的是石头，小店洗手的都是这样粗犷的石盆，整体用石头凿成，如果不是太重，也想买个回去用，可以用几代人了。

![]( {{ site.url }}/assets/blog-images/20161002/a-54.jpg)

店家非常贴心地上了第一道菜，炒粉干，好像是番薯粉干，半透明

![]( {{ site.url }}/assets/blog-images/20161002/a-55.jpg)

半小时后，风卷残云般一扫而光，才想起没有拍下菜肴，我们点了鲳鱼、豆腐鱼、梅头鱼、虾、乌贼、炒粉干、茭白、番茄汤，只花了185，真是便宜。而且都很新鲜，味道很好，看看光盘就知道了，出去游玩吃成光盘的时候不多。

![]( {{ site.url }}/assets/blog-images/20161002/a-56.jpg)

吃完和老板聊起马岙潭的经历，老板说边上就有个沙滩，也很干净，也是退潮才露出来，算算时间，现在正是退潮的时候。关键是不远，只有十几分钟的路程，于是我们就开车来到了西浪村。

这里也有许多车停在路边，但是比马岙潭要好点，我把车停到了一户人家的菜地里，想付点停车费给主人家，那人连连摇手，连说不用，停停没事。这和舟山的沙滩停车成鲜明对比，舟山是本地人在路边拉客停车，可能也和这里是不知名小岛，来的都是乐清本地人有关吧。

这里的地形也是山崖沙滩，远眺出去，停着几艘船，不知道是什么船

![]( {{ site.url }}/assets/blog-images/20161002/a-57.jpg)

在饭桌上，不知怎么的聊到了河里发生一些怪事。我说小时候记得河里有几回淹死人，老爸就打开话匣子了，他说了三件事。

第一件事是他小时候和小伙伴去游泳，共有五个人，结果他跳水进去，腿上就被磕破了个大口子，跑回家包扎去了。在岸边的一个伯父发现只有四个小孩，吓了一大跳，说天天五个小孩一起玩水，今天刚刚看到五个在玩，怎么一转眼只有四个了，紧张的下水去摸，然后很多人就下去摸。爸爸他回家包扎好后，回到河边，看到这么多人在摸，好奇地凑过去问，你们在摸什么，伯父一抬头说，原来你在呀，还以为你淹水里了。此后一段时间，他们几个小孩不敢再下水，虽然没出事，但是心里总是怕怕的。

第二件事是希放哥小时候时住在河浃头那边，一年发大水，路和河分不清了，希放哥不小心掉到河里，希平妈急死了，几个人把希放哥救上来，发现没气了。他妈妈把他放在倒扣的锅底上，然后人工呼吸救活了。

第三件事就有点灵异了，说早三十年前，村边河畔有个养鸭子的，半夜发现有个贼，跑去追，那贼跑在河岸上，一个不小心，跘在石头上，钻河里去了，他守了半天没见上来，就用石头在岸边做了个记号回去睡觉了。第二天，跟村里人说起，村里人说那个贼老早在他看不到的地方上岸了，贼的水性都好的很，但是他坚持说就是这里掉下去的，没见起来，肯定在里面。于是几个胆子大的下去摸，其中就有一个家边的邻居，胆子大的很，按老爸的话说，是多么勇猛的一个人。结果几个人摸不着，正准备走时，那尸体自己浮上来了，原来是栽在淤泥里，他们摸来摸去，搅动水就浮上来了，当场几个摸的人就吓住了，还真有哇。那个很勇猛的，吓得直接在河东上了岸，我们村是在河西，他宁可绕远路过桥，也不敢下水过河。
